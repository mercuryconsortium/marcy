SSH on Windows¶

Windows users must use a secure shell client. PuTTY is the recommended program for this task, so we will this program in the instructions below.

Download and install PuTTY on your local computer (localhost). Open a new SSH connection and edit the prompt accordingly. Enter marcy.furman.edu in the Host Name box and click Open. A command line terminal will open and ask for your account credentials. Enter your username and password to gain access to Marcy.
